# Power-BI-shipped-vs-sold-dashboard
Supply chain analytics dashboard built with Power BI to track sold vs shipped items and monitor order fulfillment performance.

## Project Overview
The dashboard compares sold items and shipped items to analyze order fulfillment performance and identify potential shipping gaps.

## Objectives
- Track total sold vs shipped items over different months
- Analyze shipment performance across different markets (countries)
- Provide interactive filters to explore sales and shipping data by region and time period

## Tools Used
- Power BI
- Excel / CSV dataset

## Skills Demonstrated
- Data Visualization
- Supply Chain Analytics
- Dashboard Design
- KPI Analysis

## Key Metrics
- Total Items Sold
- Total Items Shipped

## Business Value
This dashboard enables businesses to monitor supply chain performance, identify gaps between sold and shipped items, ensure that the balance between sales and shipments is properly managed, analyze sales and shipment trends, and support data-driven logistics decisions.
